# TEMPLATE — tiêu đề ngắn cho bài thực hành

> Hãy thay toàn bộ nội dung file này khi bạn sao chép template.

## Mục tiêu

Một câu mô tả những gì học viên sẽ đạt được sau khi hoàn thành bài thực hành.

## Yêu cầu

- Bạn đang làm việc bên trong Codespace của repo này, nên Vault dev server đã
  được khởi động sẵn ở `http://127.0.0.1:8200` với root token là `root`.
- Bạn đã đọc bài lý thuyết tương ứng trong `site/docs/`.

## Nhiệm vụ của bạn

1. Bước một — mô tả ngắn gọn việc cần làm.
2. Bước hai.
3. Bước ba.

> Gợi ý: hãy tự suy nghĩ trước khi mở `solution.md`. Nếu bí, đối chiếu với phần
> giải đáp.

## Tiêu chí thành công

Chạy bộ kiểm tra:

```bash
bash verify.sh
```

Bạn phải thấy `[PASS]` cho từng kiểm tra và dòng cuối `Tất cả kiểm tra đều đạt.`

## Cần đáp án?

Mở [`solution.md`](./solution.md) để xem lời giải mẫu kèm giải thích.
