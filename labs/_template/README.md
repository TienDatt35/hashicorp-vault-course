# TEMPLATE — tiêu đề ngắn cho bài thực hành

> Hãy thay toàn bộ nội dung file này khi bạn sao chép template. Phần front-matter
> sẽ được widget `<LabCallout/>` đọc trên trang lý thuyết tương ứng.

## Mục tiêu

Một câu mô tả những gì học viên sẽ đạt được.

## Yêu cầu

- Bạn đã mở thư mục này trong một Codespace (`make setup` đã tự chạy).
- Bạn đã hoàn thành bài học lý thuyết tương ứng.

## Nhiệm vụ của bạn

1. Bước một.
2. Bước hai.
3. Bước ba.

## Tiêu chí thành công

Khi chạy `make verify`, bạn phải thấy `[PASS]` cho từng kiểm tra và dòng cuối
"All checks passed."

## Hiện đáp án

<details>
<summary>Hiển thị đáp án mẫu</summary>

Mô tả lời giải chuẩn ở đây, sau đó chạy:

```bash
make solution
```

Lệnh này áp dụng `solution/apply.sh` rồi chạy lại `verify.sh`.

</details>
