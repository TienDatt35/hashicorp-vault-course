# Chào mừng đến với khóa học Vault Associate (003)

Khóa học này giúp bạn chuẩn bị cho kỳ thi **HashiCorp Certified: Vault Associate (003)** thông qua:

- **Bài học lý thuyết** với sơ đồ tương tác, cây quyết định và bài kiểm tra (chính bạn đang đọc một bài như vậy).
- **Bài thực hành** chạy trên GitHub Codespaces với một Vault server thật. Chỉ cần bấm nút **Mở trong Codespaces** ở bất kỳ trang bài thực hành nào, bạn có thể gõ lệnh chỉ trong vài giây — không cần cài đặt gì lên máy tính cá nhân.

## Cấu trúc một chương

1. Đọc các trang lý thuyết.
2. Sử dụng các widget tương tác để kiểm tra hiểu biết.
3. Mở bài thực hành tương ứng trong Codespaces.
4. Hoàn thành bài tập rồi chạy `make verify` để kiểm tra đáp án.
5. Làm bài kiểm tra cuối chương.

Sẵn sàng chưa? Hãy mở mục **Kiến thức nền tảng về Vault** ở thanh điều hướng bên trái.
